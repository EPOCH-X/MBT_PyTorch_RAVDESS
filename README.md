# MBT_PyTorch_RAVDESS
Unofficial PyTorch implementation of "Attention Bottlenecks for Multimodal Fusion" (NeurIPS 2021). Tested on the Kaggle RAVDESS dataset for multimodal emotion classification. "Attention Bottlenecks for Multimodal Fusion" (NeurIPS 2021) 논문의 비공식 PyTorch 구현체입니다. Kaggle RAVDESS 데이터셋을 통해 멀티모달 감정 분류 성능을 테스트했습니다.
